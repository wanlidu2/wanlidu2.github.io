SQL essential training:
1 SQL overview
SQL is designed for specific tasks with related databases. It’s a single language with a single syntax. A statement is a unitive execution in SQL.
The SELECT statement retrieves the rows from the table and the WHERE clause represents the comparison. It often includes one or more clauses, for example, the FROM clause which specifies the table that the SELECT statement will use., and the WHERE clause specifies which rows will be selected by using a logical expression.
The executive sentence usually ends with a semicolon. Besides, the white space is ignored.
The fundamental functions are create, read, update and delete.
SELECT is used for all queries that return values and retrieve information from the database.’*’ is used to get all the rows and columns. It is used to retrieve data from a SQL database.
COUNT functions can count rows without listing them after the SELECT clause.

INSERT statement is used to add a row to a table that can specify the table and the columns, then the data will be filled in those columns. In order to make the code more readable, we can indent the second line to indicate it’s part of the same statement.
The UPDATE statement is used to change data with SET and WHERE to select the place to be updated.
DELETE is used to remove rows from a table.
Database Organization:
A relational database has many tables with rows and tables. A unique key identifies a table row and sometimes the unique key is hidden but the table must have one. This unique key is also called the primary key which means that the column is used as the unique key for this table. The tables’ keys are used to create some relationships between tables. If one table has keys that refer to the keys of other tables, they are called foreign keys.
We can press F9 to execute the query as well as the run button.
It is not necessary to capitalize the SQL keywords, but it’s a good way to set the command apart, parse a complicated query and make the code more readable.
A single quote is used for literal strings in SQL, while double quotes are used to set an identifier which allows empty space in it.
AS is also a keyword to set the column name which creates an alias identifier for the column. Sometimes the AS keyword is omitted but it still works. Usually, we enter the whole clause for legibility. We can change the order of the headings by changing the order of the words after the AS commands. Besides., we can choose which columns to return by listing columns in the selected statement.
ORDER BY clause can display the table in a specific value.
LIMIT can restrict the rows in the result and OFFSET helps to show the next n value after the first setting number. LIMIT and OFFSET need to be last and FROM needs to be first.
Exercises:
You must execute the select statement after the insert into statement to see the result with the added row. The inserted columns and values must be placed inside parentheses. The unpopulated columns are given a null value.
The DELETE FROM statement removes rows from a table and once a row has been deleted it cannot be recovered. The DELETE FROM statement is not used to remove columns from a table. The asterisk is not necessary for the code to remove a row from a table.
You can query columns in any order.???? With a decent query optimizer: it shouldn’t.
You can only tell for your cases by measuring. And the measurements will likely change as the distribution of data changes in the database.
